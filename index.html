Week 1

Progress In week 1, my group and I have decided on creating a wristband with gps to tackle the challenge statement related to the security industry, so that the security guard's prowl path may be recorded. We did further research and listed down resources, hardware(gps tracker) and software (MSSQL, MYSQL, C#) that we intend to use for this project. We also discuss on how we each can contribute to this project with our strength. Contributions I suggested some ideas for example by using a mobile app. Users can use the app gps map to track where is the security guard's patrol path. In any emergency, users can use the emergency function to ping to the security guard's wristband to alert that there is danger at the area, and need the guard to be there to assist.
